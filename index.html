About Super ZEE

Super ZEE is inspired by my personal journey in the ALX Software Engineering program. I wanted to create a fun, interactive platformer game to represent each sprint I completed during the program. As a gamer and aspiring developer, Super ZEE combines my love for video games with the skills I've learned. This portfolio project is a tribute to my growth and determination throughout ALX.

This project is a portfolio submission for the ALX Software Engineering Foundations program.
